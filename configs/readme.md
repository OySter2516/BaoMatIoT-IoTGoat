# CẤU HÌNH

Thư mục này lưu trữ các tệp cấu hình được thu thập từ firmware OWASP IoTGoat.

Các tệp cấu hình phục vụ cho quá trình phân tích hệ thống, đánh giá dịch vụ và xác định các thành phần có thể ảnh hưởng đến an toàn thông tin.

Các cấu hình được thu thập trực tiếp từ môi trường triển khai cục bộ.
